# Elements dans un contrat norme ERC721

Ici la liste des elements **obligatoires** et **facultatifs** mais utiles d'un contrat qui respecte la norme ERC721.

Lien vers la doc : https://eips.ethereum.org/EIPS/eip-721

## ERC-721 Obligatoire

| Fonction / Événement          | Description |
|-------------------------------|-------------|
| `balanceOf(address _owner)`   | Retourne le nombre de NFT possédés par `_owner`. |
| `ownerOf(uint256 _tokenId)`   | Retourne l’adresse du propriétaire du NFT `_tokenId`. |
| `safeTransferFrom(address _from, address _to, uint256 _tokenId)` | Transfert sécurisé d’un NFT avec vérification si `_to` est un contrat. |
| `safeTransferFrom(address _from, address _to, uint256 _tokenId, bytes data)` | Transfert sécurisé avec données supplémentaires. |
| `transferFrom(address _from, address _to, uint256 _tokenId)` | Transfert direct d’un NFT. |
| `approve(address _approved, uint256 _tokenId)` | Approuve `_approved` à gérer `_tokenId`. |
| `setApprovalForAll(address _operator, bool _approved)` | Approuve ou révoque `_operator` pour gérer tous les NFTs du propriétaire. |
| `getApproved(uint256 _tokenId)` | Retourne l’adresse approuvée pour `_tokenId`. |
| `isApprovedForAll(address _owner, address _operator)` | Vérifie si `_operator` est approuvé pour tous les NFTs de `_owner`. |
| `Transfer(address indexed _from, address indexed _to, uint256 indexed _tokenId)` | Événement déclenché lors d’un transfert. |
| `Approval(address indexed _owner, address indexed _approved, uint256 indexed _tokenId)` | Événement déclenché lors d’une approbation. |
| `ApprovalForAll(address indexed _owner, address indexed _operator, bool _approved)` | Événement déclenché lors d’une approbation globale. |

## ERC-721 Optionnel : Metadata

| Fonction          | Description |
|------------------|-------------|
| `name()`          | Retourne le nom de la collection de NFT. |
| `symbol()`        | Retourne le symbole de la collection. |
| `tokenURI(uint256 _tokenId)` | Retourne l’URI d’un NFT spécifique. |

## ERC-721 Optionnel : Enumeration

| Fonction          | Description |
|------------------|-------------|
| `totalSupply()`               | Retourne le nombre total de NFTs émis. |
| `tokenByIndex(uint256 _index)`| Retourne l’ID du NFT à l’index `_index`. |
| `tokenOfOwnerByIndex(address _owner, uint256 _index)` | Retourne l’ID du NFT de `_owner` à l’index `_index`. |
